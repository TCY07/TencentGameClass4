# TencentGameClass4
腾讯游戏客户端公开课作业4

选取的demo是ue4自带的示例游戏Advanced Vehicle  
删除了该demo中自带的少量UI界面，去掉了部分不必要功能（如声音效果）  
新增游戏功能：  
1.得分：每次驾驶车辆经过一道门时得一分。场景中共有2道门，需要交替经过，连续经过同一道门只得一分。  
2.计时：计算开始游戏或者Restart后的总用时。  
设计了游戏UI：  
1.显示车速、挡位。  
2.显示得分和用时。  
3.添加“暂停”按钮，在暂停菜单可以选择“恢复游戏”，“清空得分和计时”，以及“退出游戏”三个选项。
